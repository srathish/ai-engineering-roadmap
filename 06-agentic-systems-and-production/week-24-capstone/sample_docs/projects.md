# Projects Overview

## Project Alpha
Project Alpha is our flagship customer-facing dashboard rewrite. It ships on
2026-07-15. The approved budget is $42,000. The technical lead is Priya, and the
project depends on the new authentication service landing first.

## Project Beta
Project Beta is a data-export feature. It is currently on hold pending legal
review of the data-retention requirements. The estimated cost is $18,500. No ship
date has been set.

## Project Gamma
Project Gamma is an internal analytics tool. It is fully funded at $30,000 and is
expected to take roughly four months. Gamma reuses the charting library built for
Project Alpha.
